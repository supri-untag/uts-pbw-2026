Program To Do List - What Me Do

Untuk menyelesaikan Tugas UTS - Supriyanto (A18.2025.00197)

Deskripsi Program To Do List

Program ini digunakan untuk mencatat daftar kegiatan atau pekerjaan yang harus dilakukan. Pengguna dapat menambahkan nama kegiatan dan tanggal kegiatan melalui form yang tersedia.

Program ini dibuat untuk menyelesaikan Tugas UTS - Supriyanto (A18.2025.00197).

Data yang telah disimpan akan tampil pada tabel hasil input. Setiap data dapat diberi tanda selesai, diedit, maupun dihapus.

Fitur Program
  • Menambahkan data to do list
  • Menampilkan data dalam tabel
  • Memberi tanda selesai dengan checkbox
  • Mengedit data kegiatan
  • Menghapus data kegiatan
Program dibuat menggunakan HTML, CSS Bootstrap 5, dan JavaScript.
